# Expected Output Description

The sample command should produce a PDF with:

- A4 portrait pages.
- A designed cover page with title, author, date, and document type.
- A generated table of contents.
- Clear H1/H2/H3 hierarchy.
- Styled quote and callout cards.
- A styled comparison table.
- A responsive SVG figure with caption.
- A dark code block with a language label.
- Subtle page header, footer, and page number styling.
- Passing basic PDF QA.
